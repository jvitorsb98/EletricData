# EletricData

O EletricData é um programa que oferece uma interface amigável para visualizar os dados de saída do programa PCFLO e facilitar a análise de sistemas elétricos de potência.

## Funcionalidades

- Importação dos dados de saída do programa PCFLO.
- Visualização de dados em formato tabular.
- Funções de filtragem dos dados para facilitar a análise.
- Exportação dos dados para formatos populares, como CSV, Excel e PDF.

## Requisitos do Sistema

- Sistema operacional Windowns

## Instalação

1. Faça o download do instalador.
2. Execute o instalador e siga as orientações.
